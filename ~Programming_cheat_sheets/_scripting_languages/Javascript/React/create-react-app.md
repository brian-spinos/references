# React create-react-app (CLI)

```bash
$ nvm use 6.9.0

$ npm install -g create-react-app
$ create-react-app brian-app

$ cd brian-app
$ npm start # opens the browser in localhost:3000

$ npm run build # to create a build for the server

$ npm install -g serve # for a server that servers your app!

# It servers your build in (http://localhost:5000/) 
# and in your local network (http://192.168.0.2:5000/)
$ serve -s build  

```

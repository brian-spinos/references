# ZSH and themes

```bash
# how to install:
https://www.youtube.com/watch?v=KUKAKX26anA


instal zsh with brew


# change the shell
sudo chsh -s $(which zsh)


# install oh-my-zsh
https://github.com/robbyrussell/oh-my-zsh
https://github.com/robbyrussell/oh-my-zsh/tree/master/themes # themes


# zsh configuration
~/.zshrc


# to use agnoster theme: (user powerline)
https://github.com/powerline/powerline

pip install --user powerline-status


# now get the fonts
https://github.com/powerline/fonts
https://github.com/powerline/fonts/tree/master/Inconsolata # choose the .otf



# install powerlevel9k theme!
https://github.com/bhilburn/powerlevel9k
$ git clone https://github.com/bhilburn/powerlevel9k.git ~/.oh-my-zsh/custom/themes/powerlevel9k



# customization of the prompt
https://github.com/bhilburn/powerlevel9k/wiki/Show-Off-Your-Config
#==> paste the code in ~/.zshrc file, at the bottom! (for the theme you want)
```
